# Contributing

Thank you for your interest in contributing to NarutoStorm2Revived.

## How to Contribute

1. Fork the repository.
2. Create a feature branch.
3. Make your changes.
4. Test your changes.
5. Submit a Pull Request.

## Areas We Need Help With

- Reverse engineering
- Asset research
- Documentation
- UI improvements
- Debugging tools
- Stability fixes
- Modding support

## Rules

- Do not submit copyrighted game files.
- Do not submit leaked content.
- Keep changes documented.
- Test before submitting.
