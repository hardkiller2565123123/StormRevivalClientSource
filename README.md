
> **NarutoRevival Offline Release**
>
> This repository contains the source code for all of my Naruto Storm projects. These releases are currently bare-minimum builds designed to get each game launching successfully and reaching the Online Menu. They serve as the foundation for future development, framework improvements, and additional features planned for the NarutoRevival project lineup.
>
> Supported projects include multiple entries from the Naruto Storm series, which has grown into one of the most successful Naruto game franchises. ([Naruto Official][1])
>
> **Current Features**
>
> * Game launch support
> * Basic framework initialization
> * Online Menu access
> * Project-specific startup hooks
> * Foundation for future offline/LAN functionality
