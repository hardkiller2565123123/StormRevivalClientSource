# NarutoRevival Offline Release

This repository contains the source code for all NarutoRevival projects. These releases are currently early-stage offline builds intended to provide a foundation for future development, research, and preservation efforts. The primary goal of this project is to document, research, and better understand the online systems, networking, and internal functionality of the Naruto Storm series.

NarutoRevival is an open-source preservation and community project focused on bringing long-term support, tooling, documentation, and eventually online functionality back to multiple entries in the Naruto Storm franchise.

## Project Status

The current releases should be considered research and development builds. They are not representative of the final vision for the project and are primarily intended for testing, reverse engineering, and framework development.

### Current Features

* Game launch support
* Basic framework initialization
* Project-specific startup hooks
* Core project infrastructure
* Online Menu access where currently supported
* Reverse engineering and research framework
* Foundation for future LAN and online functionality
* Shared framework architecture for future expansion

### Important Notes

* These builds do **not** currently provide LAN or online play functionality.
* The primary purpose of these releases is research, reverse engineering, framework development, testing, and preservation.
* Features and compatibility will continue to improve over time.
* Some projects may be incomplete, experimental, or unstable during development.
* Documentation may be incomplete while research is ongoing.

## Future Plans

NarutoRevival is being developed as a long-term open-source project.

Planned goals include:

* Dedicated API support for each game
* Master server infrastructure
* Matchmaking systems
* Leaderboards and player statistics
* Profile systems
* LAN functionality
* Community tools and utilities
* Expanded modding support
* Complete technical documentation
* Research documentation and findings
* Unified development framework across all supported games

## Open Source Commitment

NarutoRevival will always remain open source. The source code, documentation, research findings, and project resources will be publicly available for the community to learn from, contribute to, and help preserve the Naruto Storm series for future players and developers.

Thank you for supporting NarutoRevival and helping shape the future of these projects.
