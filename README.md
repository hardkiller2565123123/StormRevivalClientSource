# NarutoRevival Offline Release

This repository contains the source code for all NarutoRevival projects. These releases are currently early-stage offline builds designed to provide a working foundation for future development. The primary goal at this stage is to ensure each game launches successfully and reaches its highest currently supported menu state.

NarutoRevival is an open-source preservation and community project focused on bringing long-term support, tooling, and online functionality back to multiple entries in the Naruto Storm series.

## Project Status

The current releases are considered foundation builds and are not representative of the final vision for the project.

### Current Features

* Game launch support
* Basic framework initialization
* Project-specific startup hooks
* Core project infrastructure
* Online Menu access where currently supported
* Foundation for future LAN and online functionality
* Shared framework architecture for future expansion

### Current Compatibility

| Game                     | Status                               |
| ------------------------ | ------------------------------------ |
| Naruto Storm 1           | Reaches Online Menu                  |
| Naruto Storm 2           | Reaches Online Menu                  |
| Naruto Storm 3           | Reaches Online Menu                  |
| Naruto Storm Revolution  | Currently crashes                    |
| Naruto Storm 4           | Reaches Online Menu                  |
| Naruto Storm Connections | Does not currently reach Online Menu |

### Important Notes

* These builds do **not** currently provide LAN or online play functionality.
* The primary purpose of these releases is framework development, research, testing, and preservation.
* Features and compatibility will continue to improve over time.
* Some projects may be incomplete or unstable during development.

## Future Plans

NarutoRevival is being developed as a long-term open-source project.

Planned goals include:

* Dedicated API support for each game
* Master server infrastructure
* Matchmaking systems
* Leaderboards and player statistics
* Profile systems
* LAN functionality
* Community tools and utilities
* Expanded modding support
* Documentation for every project
* Unified development framework across all supported games

## Open Source Commitment

NarutoRevival will remain open source. The source code, documentation, and project resources will be publicly available for the community to learn from, contribute to, and help preserve the Naruto Storm series for future players.

Thank you for supporting NarutoRevival and helping shape the future of these projects.
